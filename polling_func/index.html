betfsm.blackboard_polling_func(path)

Returns a polling function that checks whether one of the given events are present in the blackboard at location path.

Parameters:
  • path (str) –

    location in the blackboard as a '/' separated path.
Returns:
  • Callable[[Dict, List[str]], str | None] –

    polling function

betfsm.ctrl_c_polling_func(value='CTRL_C', repeated=3)

Returns a polling function that checks whether one of the given events are present in the blackboard at location path.

Configures the system SIGINT (Ctrl+C) handler to interact with a blackboard, where a variable will set to given value if Ctrl-c is pressed

Parameters:
  • value (str, default: 'CTRL_C' ) –

    Value that will be set when ctrl-c is pressed.

  • repeated ((int, None), default: 3 ) –

    number of times to repeat ctrl-c before KeyboardInterrupt is raised
Returns:
  • Callable[[Dict, List[str]], str | None] –

    polling function

betfsm_crospi.crospi_polling_func(node=None, topic='crospi_node/my_output', queue_size=10, max_age=0.05)

Returns a callback function to read crospi events suitable for use withy betfsm.Event. Uses betfsm_ros.EventQueueSubscriber for the qeueu. The default parameters are suitable for crospi.

Parameters:
  • node (Node, default: None ) –

    ROS2 node

  • topic_name –

    name of the topic to subscribe to

  • queue_size (int, default: 10 ) –

    size of the queue (related to maximum concurrent events, i.e. sample time in relation to the events generated)

  • max_age (float, default: 0.05 ) –

    maximum age of the events that still will be received.
Returns:
  • Callable[[Dict, List[str]], str | None] –

    Polling function

betfsm.combine(*args)

Combines multiple polling callbacks, takes as arguments the functions themselves and returns a callback function that combines the polling functions.

e.g. combine(polling_func1,polling_func2,....)

The earlier arguments have priority
