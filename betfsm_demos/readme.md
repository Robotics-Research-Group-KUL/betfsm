# BeTFSM Demos package

A small set of  BeTFSM / Crospi demo applications, with two goals in mind:
- demonstrate the capabilities of BeTFSM to coordinate Crospi
- serve as an example of how you can package your own tasks/skills 

See the tutorial section of the BeTFSM documentation for more explanation on the demos.

You can run the demos using: `ros2 run betfsm_demos ....`

Demos are:
  - skill_example_1      
  - skill_example_2      
  - skill_example_3      
  - skill_example_4 
  - example_with_output  
  - example_local_task
